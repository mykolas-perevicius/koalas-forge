# ⌨️ Keyboard Shortcuts

Koala's Forge supports keyboard shortcuts for efficient navigation and control.

## Global Shortcuts

| Shortcut | Action |
|----------|--------|
| <kbd>/</kbd> | Focus search bar |
| <kbd>Esc</kbd> | Close modals / Clear search |
| <kbd>Ctrl</kbd> + <kbd>K</kbd> | Open command palette (future) |

## Navigation

| Shortcut | Action |
|----------|--------|
| <kbd>Tab</kbd> | Navigate between elements |
| <kbd>Shift</kbd> + <kbd>Tab</kbd> | Navigate backwards |
| <kbd>Enter</kbd> | Activate focused element |
| <kbd>Space</kbd> | Toggle checkboxes |

## App Selection

| Shortcut | Action |
|----------|--------|
| <kbd>Ctrl</kbd> + <kbd>A</kbd> | Select all visible apps |
| <kbd>Ctrl</kbd> + <kbd>D</kbd> | Deselect all apps |
| <kbd>Ctrl</kbd> + <kbd>I</kbd> | Invert selection |

## Mode Switching

| Shortcut | Action |
|----------|--------|
| <kbd>1</kbd> | Switch to Wizard Mode |
| <kbd>2</kbd> | Switch to Expert Mode |

## Categories

| Shortcut | Action |
|----------|--------|
| <kbd>Alt</kbd> + <kbd>1</kbd> | Show all apps |
| <kbd>Alt</kbd> + <kbd>2</kbd> | Show AI & ML apps |
| <kbd>Alt</kbd> + <kbd>3</kbd> | Show Development apps |
| <kbd>Alt</kbd> + <kbd>4</kbd> | Show Creative apps |
| <kbd>Alt</kbd> + <kbd>5</kbd> | Show Gaming apps |

## Installation Controls

| Shortcut | Action |
|----------|--------|
| <kbd>Ctrl</kbd> + <kbd>Enter</kbd> | Start installation |
| <kbd>Ctrl</kbd> + <kbd>P</kbd> | Pause/Resume installation |
| <kbd>Ctrl</kbd> + <kbd>Q</kbd> | Cancel installation |
| <kbd>Ctrl</kbd> + <kbd>T</kbd> | Toggle dry run mode |

## Other

|----------|--------|

| <kbd>Ctrl</kbd> + <kbd>S</kbd> | Save current selection |
| <kbd>F1</kbd> | Show help |
| <kbd>?</kbd> | Show keyboard shortcuts |

## Tips

- **Search is instant**: Just start typing to filter apps
- **Multi-select presets**: Click multiple preset packs to combine them
- **Right-click context menus**: Additional options for apps
- **Drag to reorder**: Drag preset packs to reorder (future feature)

## Accessibility

Koala's Forge is designed to be keyboard-accessible:

- All interactive elements can be reached via <kbd>Tab</kbd>
- Focus indicators are visible
- Screen reader compatible
- High contrast mode supported
